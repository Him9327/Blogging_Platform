Coding is my passion

[2] Programming is the art of algorithm design and the craft of debugging errant code!

[3] Most good programmers do programming not because they expect to get paid or get adulation by the public, but because it is fun to program !

[4] If debugging is the process of removing bugs, then programming must be the process of putting them in!

[5] Any fool can write code that a computer can understand. Good programmers write code that humans can understand !
